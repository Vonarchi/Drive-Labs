# CreamCode

Generate production-grade web apps from structured specs.

## Stack
- **Frontend**: Next.js 14
- **Backend**: Node
- **Database**: Supabase

## Features
- Project wizard
- Template catalog
- Renderer
- Auth
- Download + Deploy
- Audit log

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Set up environment variables:
   ```bash
   cp .env.example .env.local
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```
